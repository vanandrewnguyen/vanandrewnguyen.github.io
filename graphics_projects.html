Computer Graphics Projects

I deeply enjoy understanding the math and design behind computer graphics models. Working with graphics allows me to shift perspectives from artist to designer to researcher, and it's one of the few areas of study I've been forced to learn major theory outside of computer science; colour theory, physics models, aesthetic design.

Rendering Engines

Raytracing in One Weekend

C++

Following Peter Shirley's "Raytracing" series, I wrote a basic raytracing engine which covers diffuse lighting, subsurface scattering, volmetric mediums and caustics, and collisions with boxes and spheres. I'm extending it with custom features such as custom scene configuration and extended ray-triangle collisions.

OpenGL Rendering

C++

OpenGL

I wrote a basic renderer using OpenGL. Project is on hiatus, but originally the plan was to link the renderer with an Arduino Lidar Scanner to process real-life mediums in a basic photoscan.

Shaders

You can click on the links below to view the real-time shader. It may take a while to load. For the full collection of my shader work, please see this page

Simple path tracer

GLSL

Multiple rays are shot on each pixel into the scene, bouncing and accumulating an output. Final output is an average of every sample.

Moana Inspired Water Terrain

GLSL

This pixel shader uses a procedural volume built on layers of perlin and sine noise. Water is simulated with refracted rays, caustics is faked through voronoise.


Himilayas Inspired Terrain

GLSL

This pixel shader uses a texture as a multiple heightmaps to build a mountainous terrain.

Volumetric Procedural Model

GLSL

This pixel shader explores the use of subsurface scattering. Light propagates in the volume, which is procedurally generated with perlin noise functions. This gives the appearance of a wax-like substance.

Game of Life

GLSL

This pixel shader makes use of shader buffers to seperate states of movement in cellular automata. My variation makes use of old-school inspired effects like vignetting.

Planet Terrain

GLSL

This pixel shader makes use of triplanar mapping to map perlin noise onto a sphere sdf, creating a planet. The atmosphere is simulated using a fake 2D bloom.
